═══════════════════════════════════════════════════════════════

🎮 CONTROLS:
───────────────────────────────────────────────────────────────
  Numbers 1-9  →  Place your piece on squares 1-9
  
  Square Layout:
  ┌───┬───┬───┐
  │ 1 │ 2 │ 3 │  (1 = top-left, 3 = top-right)
  ├───┼───┼───┤
  │ 4 │ 5 │ 6 │  (5 = center)
  ├───┼───┼───┤
  │ 7 │ 8 │ 9 │  (9 = bottom-right)
  └───┴───┴───┘
  
  D  →  AI board description (~1.5s)
  R  →  Reset game
  H  →  Help

═══════════════════════════════════════════════════════════════

🎯 WHAT HAPPENS:
───────────────────────────────────────────────────────────────
  1. Game launches with voice welcome message
  2. You play as X, computer plays as O
  3. Press numbers to make moves
  4. Voice confirms each move
  5. Press D anytime for AI help
  6. Win, lose, or tie announced by voice

  ✓ Game outcomes
  ✓ AI descriptions
  ✓ Help instructions
